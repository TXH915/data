
# 数据分析在线教育平台 - 产品需求文档

## 1. 产品概述
- 一款基于Python的数据分析在线教育平台，为商务数据分析与应用专业学生提供学习、练习和测评服务。
- 无需登录注册，直接访问即可使用。
- 部署至Cloudflare Pages。

## 2. 核心功能

### 2.1 课程体系
- **10个完整项目**，从基础入门到进阶再到高级
- 每个项目包含：
  - 概念理论模块（全面易懂的知识内容）
  - 测试代码和运行代码模块（空代码，让学生自己写）
  - 参考答案模块
  - 测评小考和练习题
- 同一页面内可以自由拉动调整各模块布局

### 2.2 互动式学习模块
- 学：理论知识学习
- 练：编程练习
- 测：测评考试

### 2.3 成就激励系统
- 完成项目获取成就
- 激励持续学习

### 2.4 Python代码调试运行
- 在线代码编辑器
- 实时运行Python代码

## 3. 项目清单

### 入门项目
1. 数据清洗实践
2. 数据可视化基础
3. 统计分析入门
4. Pandas数据处理
5. Matplotlib绘图基础

### 进阶项目
6. 购物篮分析（关联规则）
7. 时间序列分析
8. 文本分析基础

### 高级项目
9. 异常值检测
10. 机器学习入门-分类问题

## 4. 非功能需求
- 响应式设计，支持桌面端
- 优秀的视觉设计和用户体验
- 代码编辑器支持语法高亮
- 无登录注册，立即使用
